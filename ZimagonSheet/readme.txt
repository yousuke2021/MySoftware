①　挨拶

	この度は「ZimagonSHEET」をダウンロードしていだたきありがとうございます。
	このソフトウェアはZimagonカードゲームで使用するプレイマットを作るためのものです。
	WPFというフレームワークを使って開発されています。


②　動作環境

	Windows10
	
	動作させるためには「.NET Desktop Runtime 5.0.X」が必要です。
	ダウンロードリンク：https://dotnet.microsoft.com/download/dotnet/5.0


③　起動方法

	ソフトウェア本体はbinフォルダーの中に入っています。
	ZimagonSHEET.exeをダブルクリックして起動します。
